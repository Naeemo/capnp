# LTU - 产品形态详细设计

## 核心决策确认

### 1. 互动深度：混合模式（C）
**事实性问题** → 准确回答（知识库/RAG）
- "太阳是什么颜色的？" → "黄色的大火球"
- "恐龙还有吗？" → "灭绝了，现在只有化石"

**开放式问题** → 引导回具体话题
- "为什么？" → "你觉得呢？我们先看看书里的解释..."
- "然后呢？" → "故事接下来要发生一件有趣的事，想听听吗？"

**目的**：避免LLM在开放域胡说，同时保持对话感

---

### 2. 内容来源：混合模式（C）
**框架预制**（安全层）
- 故事模板（起承转合固定，角色可变）
- 游戏类型（找颜色、数数、模仿动作）
- 知识边界（只回答知识库内的内容）

**LLM填充**（灵活层）
- 角色对话语气
- 根据孩子回应调整节奏
- 生成变体（"小兔子"→"小猫咪"版本）

**技术实现**：
```
预制模板 + RAG知识库 + LLM生成（受控提示）
↓
输出过滤器（敏感词、事实核查）
↓
语音输出
```

---

### 3. 时长控制：三层机制

| 层级 | 控制方 | 机制 | 场景 |
|------|--------|------|------|
| **硬上限** | 家长 | 后台设定每日/单次最大时长 | 防止过度使用 |
| **软提醒** | AI | 主动建议休息（"我们休息5分钟，喝水去吧"） | 自然停顿点 |
| **即时控制** | 孩子 | 关闭封盖开关 = 立即停止 | 孩子不想玩了 |

**结束触发优先级**：
1. 封盖关闭（立即）
2. 家长远程终止（立即）
3. 硬上限到达（到时间后当前活动结束）
4. AI建议休息（可接受或拒绝）

## 互动场景详细设计

### 场景A：自由探索（孩子主导）
**触发**：孩子打开封盖，不说话或说"我想玩"

**AI开场**：
"你好呀！我是小星。今天想听故事，还是一起唱歌，或者你有问题想问我？"
[停顿3秒，等待回应]

**分支**：
- 孩子明确说"故事" → 进入故事库选择
- 孩子明确说"唱歌" → 进入儿歌列表
- 孩子问问题 → 进入问答模式
- 孩子不说话/说不清楚 → "那我们随便逛逛？我刚看到一只小青蛙..."

### 场景B：问答模式
**孩子提问**："为什么会下雨？"

**处理流程**：
1. 意图识别 → "天气/自然现象"类别
2. 知识库检索 → 有标准答案？
   - 有 → 简化回答（3岁版）："云宝宝里的水太多，装不下就掉下来啦"
   - 无 → "这个问题好难呀，我记下来，等妈妈回来我们一起查书好吗？"
3. 追问处理：
   - "云为什么会装水？" → 继续简化解释（最多2层）
   - "然后呢？" → 引导回故事/话题

### 场景C：故事模式
**结构**：预制模板 + 实时调整

```
[开场 - 固定]
"今天要讲小兔子找胡萝卜的故事。小兔子叫... [LLM生成名字]，它住在一个... [孩子选择：山洞/树下/花丛]"

[发展 - 分支节点]
"小兔子走到河边，想过河。你觉得它应该：
A. 走木桥  B. 游泳  C. 等大乌龟"
[摄像头识别孩子指向/说出选项]

[高潮 - 根据选择调整]
选A（木桥）→ "桥有点晃，小兔子有点害怕。你想对它说什么？"
→ 孩子回应 → LLM生成鼓励的话

[结局 - 固定框架]
"最后小兔子找到了胡萝卜，它想谢谢你。要拍照留念吗？"
[摄像头拍照，合成到故事结尾画面]
```

### 场景D：身体游戏
**类型**：模仿动作、找东西、跳舞

**示例**：
"我们来玩木头人！我说'一二三'你就动，说'木头人'你就停。准备好了吗？"
[摄像头识别动作]
"哈哈你动了！我看到了你的手在晃～"

## 技术架构概要

### 输入处理
```
摄像头视频流
├── 人脸识别 → 确认孩子在场、距离提醒
├── 动作识别 → 游戏互动、手势选择
└── 物体识别 → "这是什么"问题

麦克风音频
├── ASR（童声优化）→ 文本
├── 情绪识别（语调）→ 开心/无聊/烦躁
└── 关键词触发 → "妈妈"/"害怕"/"不玩了"
```

### 决策引擎
```
意图分类器（预制规则 + 轻量模型）
    ↓
内容路由
├── 事实问答 → RAG知识库 → 答案简化
├── 故事/游戏 → 模板引擎 → LLM填充
├── 闲聊/开放 → 引导策略 → 转回具体话题
└── 情绪异常 → 安抚策略 / 通知家长
```

### 输出生成
```
内容生成
├── 文本 → TTS（温暖童声）
├── 视觉 → 预制动效 / 简单生成
└── 反馈 → 音效 / 画面特效

安全过滤
├── 内容审查（敏感词、不当话题）
├── 事实核查（知识库比对）
└── 边界检查（不承诺、不冒充）
```
