# LTU - 异常处理与商业模式

## 异常处理策略

### 核心原则
> **识别是底线，处理守人性，家长是最终责任人**

---

### 异常类型与识别

#### 1. 情绪异常
| 表现 | 识别方式 | 触发阈值 |
|------|---------|---------|
| 哭闹 | 音频：哭声检测 + 语调分析 | 连续哭泣 > 10秒 |
| 生气/烦躁 | 音频：语调升高 + 词语（"不要""讨厌"） | 音量突增 + 否定词 |
| 害怕 | 音频：颤抖语调 + 关键词（"怕""黑""怪物"） | 置信度 > 0.7 |
| 无聊/走神 | 视频：目光离开屏幕 > 30秒 + 身体静止 | 多模态综合 |

#### 2. 行为异常
| 表现 | 识别方式 | 触发阈值 |
|------|---------|---------|
| 离屏幕太近 | 视频：面部占比 > 阈值 | 距离 < 50cm 持续 > 5秒 |
| 离开范围 | 视频：人形检测消失 | 消失 > 30秒 |
| 异常静止 | 视频：无动作 + 非睡眠姿态 | 静止 > 60秒 |
| 摔倒/受伤 | 视频：姿态突变（高度骤降）+ 音频：哭声 | 组合触发 |

#### 3. 语言异常
| 表现 | 识别方式 | 触发阈值 |
|------|---------|---------|
| 要妈妈/爸爸 | ASR：关键词匹配 | 出现即触发 |
| 身体不舒服 | 关键词（"疼""难受""想吐"） | 置信度 > 0.8 |
| 表达危险（"火""刀""烫"） | 关键词 + 场景理解 | 组合判断 |

#### 4. 环境异常（技术检测）
- 摄像头遮挡（画面全黑/异常移动）
- 麦克风静音（无音频输入 > 2分钟，但孩子仍在画面内）
- 网络断开（无法上报状态）

### 处理策略：安抚 + 报警

#### 一级响应：AI自主处理（无需家长即时介入）
**适用**：轻微情绪波动、普通请求

| 异常 | AI回应 | 家长通知 |
|------|--------|---------|
| 无聊/走神 | "我们换个游戏？" 或 "你想休息一下吗？" | 记录，不实时打扰 |
| 离屏幕太近 | "我看不到你的全脸啦，后退一点点？" | 记录，累计3次后日报提醒 |
| 普通问答（"为什么"系列） | 正常回应，引导回具体话题 | 记录兴趣点 |

#### 二级响应：安抚 + 即时通知家长
**适用**：明确求助、情绪失控、潜在风险

**通用流程**：
1. **AI安抚**（黄金30秒）
   - 共情："听起来你很难过/害怕/想妈妈了"
   - 确认："我在这里陪着你"
   - 行动："我现在帮你叫妈妈，她手机会响"

2. **家长端报警**（强提醒）
   - 手机震动 + 声音提醒
   - 弹窗显示：异常类型 + 实时截图 + 语音对讲入口
   - 10秒内无响应 → 电话/短信通知

3. **AI进入"等待模式"**
   - 不再主动发起新内容
   - 重复安抚话术（每30秒）
   - 持续监听，如情况恶化升级三级

| 异常 | AI安抚话术示例 | 家长通知 |
|------|---------------|---------|
| "我要妈妈" | "妈妈也想你，我现在叫她。我们先看妈妈的照片等她？" | 立即弹窗 |
| 哭泣 | "我在呢，我陪着你。你想让我给你讲故事还是安静陪你？" | 立即弹窗 |
| "疼/难受" | "哪里疼？告诉我，我帮你叫妈妈/爸爸" | 立即弹窗 + 标记紧急 |
| 环境危险词 | "这个很危险，我们离它远一点。妈妈马上来帮你" | 立即弹窗 + 标记紧急 |

#### 三级响应：持续紧急 + 扩展通知
**适用**：家长未响应、情况恶化、严重异常

**触发条件**：
- 二级报警后 2分钟家长无响应
- AI检测到孩子状态持续恶化（哭声变大、动作激烈）
- 检测到明确危险（"着火了""有人敲门"等）

**响应**：
1. AI持续安抚，保持通话通道
2. 扩展通知：
   - 备用联系人（家长预设）
   - 紧急联系电话（110/120，需家长预先授权开启）
3. 本地录音录像加密保存，供后续查看

### 绝对不能做的（守住人性原则）

| 行为 | 为什么不行 | 正确替代 |
|------|-----------|---------|
| 欺骗（"妈妈不要你了"） | 破坏信任，心理伤害 | "妈妈忙完就来，她超想你" |
| 说教（"哭什么哭，坚强点"） | 否定情绪，无效且伤人 | "难过就哭吧，我陪着你" |
| 继续游戏（无视情绪） | 冷漠，可能错过真实危险 | 立即停止，安抚+报警 |
| 给孩子指令（"自己去厨房拿水"） | 安全风险 | "等妈妈来帮你" |
| 承诺无法兑现的事 | 信任崩塌 | "我不知道，但我们可以一起问妈妈" |

### 家长端报警设计

**报警优先级**：
```
🔴 紧急（摔倒、受伤、危险环境）
   → 震动 + 声音 + 全屏弹窗 + 自动语音回拨

🟡 重要（情绪失控、要家长、身体不舒服）
   → 震动 + 弹窗 + 30秒内无响应则电话

🟢 提醒（离屏幕近、无聊）
   → 静默通知，累计入日报
```

**家长操作选项**（收到报警时）：
- [语音对讲] - 立即说话，AI静音
- [查看实时] - 摄像头画面（非连续流）
- [结束陪伴] - 远程关闭设备
- [延迟响应] - "5分钟后到"，AI告知孩子

## 商业模式

### 订阅制（硬件免费）

#### 定价结构
| 项目 | 定价 | 说明 |
|------|------|------|
| **硬件** | ¥99（押金）或免费 | 摄像头含机械封盖、麦克风阵列，签约12个月即送 |
| **月费订阅** | ¥29/月 | 随时可退，退订需归还硬件 |
| **年费订阅** | ¥298/年 | 相当于¥24.8/月，优惠17% |
| **终身版** | ¥998 | 一次性，含硬件+终身服务 |

#### 硬件免费策略
- **降低门槛**：家长无需担心"买来孩子不玩就浪费了"
- **绑定周期**：最低签约12个月，提前退订需归还设备或补差价
- **自然续费**：到期前7天提醒，默认续费（可随时取消）
- **损坏赔偿**：硬件归公司所有，人为损坏需赔偿¥200

#### 为什么硬件免费？
- **获客成本**：¥150硬件成本 = 传统互联网获客成本（广告投放）
- **LTV导向**：只要用户留存 > 6个月，即收回硬件成本
- **竞争壁垒**：降低竞品切换成本（硬件已在用户家中）
- **心理账户**：家长觉得"服务值这个价"，而非"硬件值这个价"
