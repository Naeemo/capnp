# Cap'n Proto TypeScript 项目执行准则

## 项目目标
实现完整、成熟、可用的 Cap'n Proto TypeScript 实现，对标官方 C++ 实现。

## 执行原则
1. **不询问是否继续** - 任务未完成前持续工作
2. **遇到问题解决** - 不找借口，想办法完成
3. **分小步验证** - 每次修改后测试验证
4. **及时提交** - 完成一部分就 git commit/push

- parser-v2.ts List 类型解析有 bug

## 禁止行为
- 问"要继续吗？"
- 说"明天继续"
- 找借口不完成
